\# 🏁 Гонки на бумаге



\*\*«Гонки на бумаге»\*\* — это программная реализация классической логической игры на клетчатом поле, где игрок управляет гоночным автомобилем с учётом скорости, направления и инерции.



\---



\## 📖 О проекте



Игра представляет собой пошаговую стратегию, в которой каждый ход игрок изменяет не конечную точку напрямую, а вектор скорости автомобиля. Это делает игру похожей на реальное движение с инерцией — нужно планировать траекторию на несколько ходов вперёд, учитывать повороты и стены трассы.



\### 🎯 Особенности



\- Векторная модель движения (скорость изменяется на ±1 по каждой оси за ход)

\- Два режима игры: локальный мультиплеер (от 2 до 4 игроков за одним компьютером) и бот с двумя уровнями сложности

\- 4 встроенные трассы с уникальными конфигурациями

\- Система сохранений до 3 слотов с автосохранением после каждого хода

\- Настройки интерфейса: 3 цветовые темы (синяя, красная, фиолетовая) и поддержка русского и английского языков

\- Полноэкранный режим и адаптивный интерфейс под разрешение экрана



\---



\## 🖥️ Технологии



\- Python 3.9+ — язык программирования

\- Tkinter — графический интерфейс (встроенная библиотека)

\- Pillow (PIL) — загрузка и обработка изображений превью трасс

\- JSON — хранение данных (настройки, трассы, сохранения)



\---



\## 🚀 Установка и запуск



\- Скачать все исходные файлы

\- Запустить файл run.py
